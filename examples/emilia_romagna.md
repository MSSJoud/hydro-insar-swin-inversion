# Emilia-Romagna example

This repository uses Emilia-Romagna, Italy, and specifically the Bologna overlap workflow, as the first example case.

The example is meant to demonstrate:

- study-area configuration
- mounted-data layout
- grouped multisensor inversion concepts
- well-validation integration

For the manuscript-facing results, figures, and detailed interpretation, see the paper companion repository:

- `https://github.com/MSSJoud/insar-layered-groundwater-mcmc-swin`

## Current example scope

The current software repository provides:

- an example config file
- container runtime support
- a base CLI for configuration checks

The full end-to-end Emilia-Romagna workflow is being migrated from the research workspace into this software structure.
